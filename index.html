Section 01
프로젝트 개요 및 배경

세이브트리&엑스퍼트는 LiDAR 3D 수목 맵핑과 IoT 실시간 모니터링을 결합해 도심 수목을 디지털 자산으로 전환하는 스마트 조경 솔루션 기업이다. 두꺼비세상·아파트너는 이 서비스의 아파트 단지 확장 파트너로, 관리자·입주민용 수목 관리 대시보드 플랫폼을 공동 개발하는 것이 본 프로젝트의 목표다.

세이브트리 하용훈 대표 공개 발언 (2024.11): "greehill, 두꺼비세상, 아파트너와의 협력을 통해 기술적 시너지를 극대화하고 글로벌 조경관리 딥테크 기업으로 성장할 것" — 공식 파트너 관계로 확인됨.
Core Tech

Greehill STI

체코 기반 딥테크. 차량 탑재 LiDAR로 수천 그루를 수일 만에 3D 맵핑. 수목별 수고·DBH·수관 면적·건강 등급 자동 산출.

IoT Layer

세이브트리 IoT 센서

수목당 토양수분·온도·전기전도도(EC)·생체전기 센서 탑재. LaaS 모델로 실시간 데이터 수집 및 나무의사 진단 연계.

Our Role

Section 02
시스템 아키텍처

본 시스템은 3-Tier 구조로 설계된다. 데이터 레이어(Greehill + 세이브트리 IoT), 백엔드 API 레이어(두꺼비세상 개발), 프론트엔드 레이어(대시보드)로 나뉜다. 포인트 클라우드 3D 뷰어는 Greehill 자체 플랫폼으로 연동(아웃바운드 링크)하며 직접 개발하지 않는다.

Section 06
외부 API 연동 방안

Greehill은 RESTful API를 통해 STI 데이터를 제공한다. 세이브트리가 Greehill의 한국 독점 파트너이므로, API 키 및 엔드포인트 접근은 세이브트리를 통해 확보한다.

Greehill STI API — 예상 엔드포인트
  • GET /sites/:id/trees — 단지 내 전체 수목 목록
  • GET /trees/:id — 개별 수목 속성 상세
  • GET /trees/:id/pointcloud — 3D 뷰어 URL 반환
  • GET /trees/:id/metrics — ESG 수치 (CO₂, 미세먼지)
  • Webhook — 건강 등급 변경 이벤트
세이브트리 IoT API — 예상 엔드포인트
  • GET /devices?tree_id=:id — 수목 연결 센서 목록
  • GET /readings/latest?device_id=:id — 최신 수치
  • GET /readings/history — 기간별 히스토리
  • WebSocket /stream — 실시간 센서 스트림
  • GET /alerts/active — 현재 활성 알림 목록
API 스펙 확인 체크리스트: ① 수목별 ID·속성 필드명 ② 포인트 클라우드 URL 패턴 ③ 센서 데이터 항목 (토양수분·온도·EC 외 추가 항목 여부) ④ 건강 등급 기준 (A~D 또는 숫자 스케일) ⑤ ESG 수치 제공 여부 및 단위 ⑥ API 인증 방식 (API Key / OAuth2)
